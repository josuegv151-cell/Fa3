¡Gracias por querer contribuir! aqui estan las reglas basicas

Pre-requisitos:

Contar con una version de Excel instalada y Funcional

Tener las macros habilitadas

Preparacion:

Daremos clic en el boton Visual Basic de la pestaña programador, se debera crear un nuevo modulo, de preferencia en el libro que estamos trabajando, 
aqui podemos empezar a a desarrollar el codigo que queramos añadir o modificar, este debe estar en lenguaje VBA.

Modificaciones:

Una vez que se tenga la seccion de codigo que se quiera agregar o modificar se debera ingresar al archivo donde esta, en este caso Macro, 
se dara clic en el boton con un lapiz para editar el archivo y se añadira la seccion de codigo deseada, una vez agregado das clic en confirmar cambios
para crear un pull request, este se revisara y se revisara si es el cambio es favorable

Se debera esperar a la revision del codigo para la aprovacion de este, de ser aprobado se hara el merge del pull request generado.

Tambien es posible agregar un archivo nuevo seleccionando el boton con el mismo nombre, esto generara tanto una rama como un pull request 
el cual se debera esperar la desicion del dueño del repositorio sobre su aprobacion
